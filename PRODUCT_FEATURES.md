# Knowledge Graph 产品功能文档

## 1. 产品概述

Knowledge Graph 是一个可视化知识图谱编辑器，允许用户通过拖拽方式创建、编辑和组织知识节点，并支持节点的分组和嵌套，以构建复杂的知识结构图。

## 2. 核心功能

### 2.1 节点管理

#### 2.1.1 NoteNode (笔记节点)
- **标题编辑**：支持双击编辑节点标题
- **内容编辑**：支持双击编辑节点内容，支持Markdown格式
- **内容展开/收缩**：可切换节点显示内容的详细程度
- **尺寸调整**：可拖拽调整节点尺寸
- **样式定制**：统一的UI设计，清晰的视觉层次

#### 2.1.2 GroupNode (群组节点)
- **容器功能**：可包含其他NoteNode
- **群组管理**：可添加、删除群组内的节点
- **边界自动计算**：根据群组内容自动调整边界
- **标题编辑**：支持双击编辑群组标题
- **尺寸调整**：可拖拽调整群组尺寸

### 2.2 节点转换

#### 2.2.1 类型转换
- **NoteNode → GroupNode**：将笔记节点转换为群组节点
- **GroupNode → NoteNode**：将群组节点转换为笔记节点
- **转换后状态保留**：转换后节点的位置、尺寸等状态得到保留
- **即时更新**：转换操作后节点视图立即更新（已修复此问题）

### 2.3 画布操作

#### 2.3.1 基础操作
- **拖拽移动**：可拖拽节点到任意位置
- **画布平移**：拖拽画布空白区域移动视图
- **画布缩放**：支持滚轮缩放，支持缩放控制

#### 2.3.2 连接功能
- **节点连接**：支持在节点间创建连接线
- **连接点管理**：节点四周提供多个连接点
- **边的类型**：支持不同类型和样式的连接线

### 2.4 分组与嵌套

#### 2.4.1 群组管理
- **嵌套支持**：支持多层嵌套群组结构
- **层级关系**：清晰的父子节点层级关系
- **边界动态计算**：群组内容变化时自动调整边界
- **嵌套布局**：支持复杂嵌套结构的自动布局

#### 2.4.2 布局策略
- **组布局**：专门针对群组结构的布局算法
- **画布布局**：适用于整个画布的布局策略
- **递归布局**：处理多层嵌套的布局算法

### 2.5 编辑与管理

#### 2.5.1 内容编辑
- **双击编辑**：双击节点进入编辑模式
- **Markdown支持**：内容编辑支持Markdown格式
- **实时预览**：编辑内容时提供实时预览效果

#### 2.5.2 历史管理
- **撤销/重做**：支持多步撤销和重做操作
- **操作记录**：记录用户的所有编辑操作
- **状态快照**：定期保存画布状态快照

### 2.6 交互控制

#### 2.6.1 工具栏功能
- **添加NoteNode**：在画布上添加新的笔记节点
- **添加GroupNode**：在画布上添加新的群组节点
- **重置视图**：将画布恢复到初始视图状态
- **清空画布**：清空所有节点和连接

#### 2.6.2 视觉控制
- **边过滤**：可控制不同类型的边的可见性
- **缩略图**：提供画布缩略图便于导航
- **网格背景**：提供点状网格背景便于对齐

### 2.7 布局与优化

#### 2.7.1 自动布局
- **智能布局**：根据节点关系自动调整节点位置
- **避免重叠**：自动调整避免节点重叠
- **视觉优化**：优化节点和边的视觉展示效果

#### 2.7.2 边优化
- **连接点优化**：根据节点位置自动优化边的连接点
- **交叉处理**：优化跨群组边的显示效果
- **动态调整**：节点移动时动态调整相关边

## 3. 用户体验特性

### 3.1 交互体验
- **拖拽友好**：所有操作都支持直观的拖拽交互
- **即时反馈**：用户操作立即得到视觉反馈
- **流畅动画**：节点转换和布局调整具有平滑动画

### 3.2 视觉设计
- **现代UI**：采用现代化的UI设计风格
- **清晰层级**：通过颜色、边框、阴影等元素体现层级关系
- **视觉引导**：连接线和箭头清晰表达节点关系

### 3.3 性能优化
- **高效渲染**：使用ReactFlow等优化渲染性能
- **状态管理**：使用Zustand优化状态更新性能
- **防抖处理**：对频繁操作进行防抖处理以优化性能

## 4. 使用场景

### 4.1 知识管理
- **个人知识整理**：整理个人学习和工作中的知识点
- **概念关系图**：绘制各类概念之间的关系图
- **思维导图**：创建非线性的思维导图

### 4.2 项目管理
- **任务分解**：分解复杂项目为可管理的任务
- **依赖关系**：可视化任务间的依赖关系
- **进度跟踪**：通过节点状态跟踪项目进度

### 4.3 教学培训
- **课程大纲**：构建课程内容的知识结构
- **概念讲解**：用图形化方式讲解复杂概念
- **学习路径**：规划学习路径和先修关系

## 5. 技术特色

### 5.1 技术栈
- **前端框架**：Next.js 16.0.1
- **UI库**：React 19.2.0
- **图编辑器**：ReactFlow
- **状态管理**：Zustand
- **样式**：Tailwind CSS + TypeScript

### 5.2 架构特点
- **组件化**：模块化组件设计
- **类型安全**：TypeScript提供完整的类型支持
- **可扩展性**：良好的架构设计支持功能扩展
