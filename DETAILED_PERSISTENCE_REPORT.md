# 知识图谱项目详细持久化需求报告

## 1. 永久化数据分类

### 1.1 画布管理数据
- **画布元信息**
  - 画布ID（唯一标识符）
  - 画布标题
  - 画布描述
  - 创建时间
  - 更新时间
  - 最后访问时间
  - 画布状态（活跃/归档/删除）
  - 用户归属信息（多用户场景）
  - 画布标签

- **画布配置信息**
  - 视口位置（x, y坐标）
  - 缩放级别
  - 主题设置（明暗模式）
  - 网格显示设置
  - 画布尺寸信息

### 1.2 画布内容数据
- **节点数据**
  - 节点ID
  - 节点类型（普通节点/群组节点/笔记节点）
  - 节点标题
  - 节点内容（文本）
  - 节点位置（x, y坐标）
  - 节点尺寸（宽度、高度）
  - 节点样式属性
  - 节点状态（选中、编辑、展开等）
  - 父节点ID（用于嵌套结构）
  - 所属群组ID
  - 节点属性（RDF属性等）
  - 节点标签
  - 节点摘要
  - 创建和更新时间

- **群组数据**
  - 群组ID
  - 群组标题和内容
  - 群组边界信息（minX, minY, maxX, maxY）
  - 群组折叠状态
  - 包含的节点ID列表
  - 父群组ID（支持嵌套）
  - 群组样式属性
  - 创建和更新时间

- **边数据**
  - 边ID
  - 源节点ID
  - 目标节点ID
  - 连接点信息（源连接点、目标连接点）
  - 边标签
  - 边样式属性
  - 边的群组归属
  - 边的附加属性（权重、强度、方向、颜色等）
  - 跨群组关系标识
  - 创建和更新时间

### 1.3 系统辅助数据
- **操作历史记录**
  - 操作类型（添加/删除/修改）
  - 操作对象ID
  - 操作前状态
  - 操作后状态
  - 操作时间戳
  - 执行用户

- **画布快照**
  - 快照ID
  - 所属画布ID
  - 快照名称
  - 快照描述
  - 快照类型（自动/手动）
  - 完整的画布状态数据
  - 创建时间

## 2. 画布管理系统设计

### 2.1 核心功能需求
- **创建画布**：用户可以创建新的画布并设置基本属性
- **查看画布列表**：展示用户所有的画布，支持按时间、名称等排序
- **打开画布**：加载指定画布的所有数据
- **重命名画布**：修改画布标题和描述
- **删除画布**：支持软删除（归档）和硬删除
- **搜索画布**：按标题、标签、内容等搜索画布
- **画布复制**：复制现有画布创建新画布

### 2.2 画布管理界面元素
- 画布缩略图预览
- 画布标题显示
- 最后编辑时间
- 画布标签显示
- 画布状态指示器
- 操作按钮（编辑、删除、复制等）

### 2.3 画布生命周期管理
- **创建时**：生成唯一ID，设置初始配置
- **编辑时**：自动保存，记录变更历史
- **关闭时**：确保数据同步保存
- **删除时**：支持恢复机制

## 3. 画布内容数据详细结构

### 3.1 节点数据结构
```
Node {
  id: string,                    // 节点唯一标识
  type: "node" | "group",        // 节点类型
  title: string,                 // 节点标题
  content: string,               // 节点内容文本
  position: {                    // 节点位置
    x: number,
    y: number
  },
  width: number,                 // 节点宽度
  height: number,                // 节点高度
  style: CSSProperties,          // 节点样式
  selected: boolean,             // 是否选中状态
  isExpanded: boolean,           // 是否展开状态
  groupId?: string,              // 所属群组ID
  parentId?: string,             // 父节点ID（嵌套用）
  attributes: Map<string, any>,  // 节点属性（RDF等）
  tags: string[],                // 节点标签
  summary: string,               // 节点摘要
  createdAt: Date,               // 创建时间
  updatedAt: Date,               // 更新时间
}
```

### 3.2 边数据结构
```
Edge {
  id: string,                    // 边唯一标识
  source: string,                // 源节点ID
  target: string,                // 目标节点ID
  sourceHandle?: string,         // 源连接点
  targetHandle?: string,         // 目标连接点
  label?: string,                // 边标签
  groupId?: string,              // 边所属群组ID
  style: CSSProperties,          // 边样式
  data: {                        // 边附加数据
    color?: string,
    strokeWidth?: number,
    isCrossGroup?: boolean,      // 是否为跨群组边
    sourceGroupId?: string,      // 源节点所属群组
    targetGroupId?: string,      // 目标节点所属群组
    weight?: number,             // 关系权重
    strength?: number,           // 关系强度
    direction?: 'unidirectional' | 'bidirectional' | 'undirected', // 方向
    customProperties?: Map<string, any>, // 自定义属性
  },
  createdAt: Date,               // 创建时间
  updatedAt: Date,               // 更新时间
}
```

### 3.3 位置数据管理
- **坐标系统**：使用笛卡尔坐标系，原点在左上角
- **位置精度**：浮点数坐标，支持精确位置
- **相对位置**：嵌套节点使用相对于父节点的位置
- **绝对位置**：存储节点在画布中的绝对坐标
- **位置更新**：拖拽操作后实时更新位置信息

## 4. 数据持久化策略

### 4.1 实时保存
- 用户操作后自动保存（防抖机制）
- 关键操作立即保存
- 离线时缓存到本地存储

### 4.2 版本管理
- 每个对象维护版本号
- 支持历史版本回溯
- 操作冲突检测和解决

### 4.3 数据备份
- 定期自动快照
- 手动创建备份点
- 支持数据导出和导入

## 5. 数据库映射建议

### 5.1 Neo4j 存储
- 节点和边的连接关系
- 嵌套结构关系
- 群组成员关系
- 图遍历相关数据

### 5.2 Elasticsearch 存储
- 节点的文本内容
- 节点的RDF属性
- 全文搜索索引
- 标签和分类信息

### 5.3 Redis 存储
- 画布元信息缓存
- 用户会话数据
- 实时协作状态
- 热门画布数据

这份详细报告涵盖了知识图谱项目中需要进行持久化存储的所有数据类型和结构，以及画布管理系统的具体需求。
