# 🚀 快速修复指南

## 问题症状

- ❌ 控制台报错：`<rect> attribute x: Expected length, "NaN"`
- ❌ 群组内的子节点无法拖拽移动
- ❌ 子节点与群组标题重叠
- ❌ 子节点飞出群组边界
- ❌ MiniMap 不显示群组内节点

## 解决方案概览

### 1️⃣ MiniMap NaN 错误 → 添加 nodeColor

```typescript
// GraphPageContent.tsx
const nodeColor = useCallback((node: ReactFlowNode) => {
  return node.type === 'group' ? '#e0e7ff' : '#93c5fd';
}, []);

<MiniMap 
  nodeColor={nodeColor}
  maskColor="rgb(240, 240, 240, 0.6)"
  pannable
  zoomable
/>
```

### 2️⃣ 子节点无法拖拽 → 设置 draggable 和 extent

```typescript
// GraphPageContent.tsx - 节点配置
return {
  ...regularNode,
  draggable: true, // ✅ 关键！
  ...(regularNode.groupId && { 
    parentId: regularNode.groupId,
    extent: 'parent' as const, // ✅ 使用 'parent'
    expandParent: true,
  }),
};

// ReactFlow 组件
<ReactFlow
  nodesDraggable={true}  // ✅ 全局启用
  nodesConnectable={true}
  // ...
>
```

### 3️⃣ 节点位置重叠 → 增加 GROUP_PADDING.top

```typescript
// useNodeHandling.ts & nodesSlice.ts
const GROUP_PADDING = { 
  top: 70,    // ✅ 从 50/60 增加到 70
  left: 20, 
  right: 20, 
  bottom: 20 
};
```

### 4️⃣ 防止 NaN → 使用 safeNumber

```typescript
// 所有文件中添加
const safeNumber = (value: any, defaultValue: number = 0): number => {
  const num = Number(value);
  return typeof num === 'number' && !isNaN(num) && isFinite(num) ? num : defaultValue;
};

// 使用示例
position: {
  x: safeNumber(node.position.x),
  y: safeNumber(node.position.y),
}
```

### 5️⃣ 坐标转换 → 相对/绝对坐标处理

```typescript
// GraphPageContent.tsx
const convertToRelativePosition = useCallback((node, parentGroup) => {
  if (!parentGroup) return node.position;
  return {
    x: safeNumber(node.position.x) - safeNumber(parentGroup.position.x),
    y: safeNumber(node.position.y) - safeNumber(parentGroup.position.y)
  };
}, []);

const convertToAbsolutePosition = useCallback((relativePos, parentGroup) => {
  return {
    x: safeNumber(relativePos.x) + safeNumber(parentGroup.position.x),
    y: safeNumber(relativePos.y) + safeNumber(parentGroup.position.y)
  };
}, []);
```

## 常见错误

### 错误 1: 忘记设置 draggable
```typescript
// ❌ 错误
return {
  ...node,
  // 缺少 draggable: true
};

// ✅ 正确
return {
  ...node,
  draggable: true,
};
```

### 错误 2: extent 设置错误
```typescript
// ❌ 错误 - 手动计算可能出错
extent: [padding.left, padding.top, maxX, maxY]

// ✅ 正确 - 让 ReactFlow 自动处理
extent: 'parent' as const
```

### 错误 3: 忘记转换坐标
```typescript
// ❌ 错误 - 直接使用相对坐标
updateNodePosition(node.id, currentNode.position);

// ✅ 正确 - 转换为绝对坐标
const absolutePos = convertToAbsolutePosition(
  currentNode.position, 
  parentGroup
);
updateNodePosition(node.id, absolutePos);
```
