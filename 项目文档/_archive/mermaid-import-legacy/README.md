# Mermaid 导入历史归档

这个目录只保存之前临时 Mermaid 导入能力的研究资料和示例脚本。

当前产品方向：Ontology Canvas / 本体建模画布。

当前状态：
- 不属于活跃前端/后端运行链路。
- 不属于当前导入架构。
- 仅作为历史参考保留。

未来如果重新设计导入能力，必须先定义本体导入契约，例如：
- `OntologyImportDraft`
- schema version
- class / relation / domain / subgraph 映射规则
- validation warnings
- command-based apply flow

Mermaid 以后最多作为可选兼容输入源重新接入，不能作为本体图的核心 schema。
